# Решения и компромиссы (ADR)

## ADR-001: SQLite для хранения ответов и базы знаний

**Контекст:** Нужно хранить ответы ассистента и структуру базы знаний для анализа.

**Решение:** SQLite, один файл `data/pass24.db`. Таблицы: `sessions`, `messages`, `audit_records`, `knowledge_categories`, `knowledge_entries`. Инициализация через `database/init_db.sh` и `schema.sql`.

**Причины:** Без внешних зависимостей, удобно для локальной разработки и анализа (sqlite3, DB Browser). При росте нагрузки можно рассмотреть миграцию на PostgreSQL.
